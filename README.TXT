TechSupport - o sistema de Suporte Técnico da DodgySoft.

Este projeto é parte do material para o livro

   Objects First with Java - A Practical Introduction using BlueJ
   Fourth edition
   David J. Barnes and Michael Kolling
   Pearson Education, 2008

Este projeto é discutido no capítulo 5.

Este projeto tem como objetivo implementar um sistema de suporte técnico para
clientes da companhia de software DodgySoft. Usuários podem descrever seus
problemas de software e receber conselhos instantaneamente!

A idéia é baseada no Eliza - um programa famoso descrito por Joseph Weizenbaum 
em 1966. (Faça uma busca por "Eliza" e "Weizenbaum" se você quiser saber mais
sobre isto.)

De fato, este sistema é muito mais primitivo que o Eliza. Mas é suficiente para
se equiparar à qualidade do suporte técnico de muitas empresas de software ;-)

Execute este programa através da classe SupportSystemMain.

Então comece a descrever seu problema, digitando na janela de terminal.

O propósito deste projeto é demonstrar e estudar bibliotecas de classes, como
ArrayList, HashMap, HashSet e Random.

Este projeto é apenas um primeiro, rudimentar, passo para a solução completa.
